# practo_dbms_mini_project

## Setup Instructions

1. **Save Project Folder Locally**
    - Clone this repository:
      ```
      git clone https://github.com/anio710/practo_dbms_mini_project.git
      cd practo_dbms_mini_project
      ```

2. **Setup Database**
    - Source the main database schema:
      ```
      source practo_full.sql
      ```
    - Source sample data:
      ```
      source sample_data.sql
      ```

3. **Run Backend**
    ```
    npm run dev
    ```

4. **Run Frontend**
    ```
    npm start
    ```
